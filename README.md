# Präsentation: Grundlagen von modernem JavaScript (ES6+)

Diese Präsentation vermittelt zentrale Grundlagen von modernem JavaScript ab **ES6 (ECMAScript 2015)** und späteren Erweiterungen. 

## Rahmenbedingungen

- **Anzahl Personen:** 2  
- **Dauer:** ca. 20 Minuten

## Inhalt der Präsentation

Die Präsentation ist in zwei thematische Bereiche unterteilt: grundlegende ES6-Features und weiterführende moderne JavaScript-Funktionen.

### Grundlegende ES6+ Features

Folgende Sprachfeatures werden vorgestellt und kurz erklärt:

- **let / const***
  Ersetzen var zur Deklaration von Variablen

- **Arrow Functions**  
  Eine Schreibweise für Funktionen, die besonders häufig bei Callbacks und funktionaler Programmierung verwendet wird.

- **Destructuring**  
  Ermöglicht das Extrahieren von Werten aus Arrays oder Objekten in einzelne Variablen.

- **Spread- und Rest-Operator**  
  Arbeiten mit Arrays und Objekten, z. B. zum Kopieren, Zusammenführen oder Sammeln von Werten.

- **Default Parameters**  
  Erlaubt das Definieren von Standardwerten für Funktionsparameter.

### Erweiterte ES6+ Features

Zusätzlich werden einige häufig genutzte moderne Konzepte behandelt:

- **Template Literals**  
 String-Syntax mit Variableninterpolation und mehrzeiligen Strings.

- **Promises und async/await**  
   Möglichkeiten zur Behandlung von asynchronem Code.

- **Modules (import/export)**  
  Strukturierung von JavaScript-Code durch modulare Aufteilung in mehrere Dateien.

- **Array-Methoden (map, filter, reduce)**  
  funktionale Methoden zur Verarbeitung und Transformation von Arrays.

## Ziel der Präsentation

Die Präsentation soll einen **Überblick über moderne JavaScript-Features** geben und zeigen, warum diese Konzepte heute fester Bestandteil moderner Webentwicklung sind.
